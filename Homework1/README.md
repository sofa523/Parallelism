# Описание задания
Заполнить массив float/double значениями синуса (один период на всю длину массива), количество элементов 10^7. Посчитать сумму и вывести в терминал.

# Выбор типа данных
Автоматически используется float. Использование тип данных double осуществляется с помощью переключения флага use_float=ON

# Вывод
Type array: float
Sum = 0.173887

Type array: double
Sum = 4.00163e-10
